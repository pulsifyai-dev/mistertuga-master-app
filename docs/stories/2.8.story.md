**Epic:** EPIC-2 (Post-Migration New Features)

**Priority:** P3 (future phase)
**Status:** Draft
**Agent:** @dev (Dex)
**Blocked By:** Story 2.4

---

## Story

**As an** admin user,
**I want** to upload the supplier's cost document and automatically compare it against my system-calculated costs,
**So that** I can identify discrepancies and ensure I'm not overpaying.

---

## Context

The Master Costs Excel already tracks "Gap" between system-calculated costs and Chinese supplier invoices. This story automates that comparison. The supplier sends their own cost document (Excel), and the system compares:
- Per-order values (system vs supplier)
- Total costs (system vs supplier)
- Flags mismatches above a configurable threshold

This is a P3 feature — not required for MVP. Build only after Stories 2.3 and 2.4 are complete and validated.

**Reference:** `docs/product/mistertuga_master.md` §7
**Data source:** `docs/product/🔴 Misterfutbol_Master_Costs.xlsx` (reconciliation structure)

---

## Acceptance Criteria

- [ ] **AC1:** Upload supplier document
  - Upload button on Supplier Exports page
  - Accept: .xlsx, .csv
  - Store in Supabase Storage
  - Parse uploaded file to extract: order reference, production cost, shipping cost, total

- [ ] **AC2:** Column mapping configuration
  - On upload, show detected columns
  - Allow user to map: order column, production cost column, shipping cost column, total column
  - Save mapping for reuse with same supplier format

- [ ] **AC3:** Reconciliation engine
  - Match supplier orders to system orders by order_number
  - Calculate per-order gap: (system_cost - supplier_cost)
  - Calculate total gap: sum of all per-order gaps
  - Flag mismatches above threshold (configurable, default: $5/order or 10%)

- [ ] **AC4:** Reconciliation results table
  - Columns: Order #, System Cost, Supplier Cost, Gap, Status (Match/Mismatch)
  - Summary row: Total System, Total Supplier, Total Gap
  - Highlight rows with mismatches (red/amber)
  - Filter: show all / mismatches only

- [ ] **AC5:** Reconciliation history
  - Store reconciliation results in database
  - View past reconciliations: date, supplier file, total gap, status
  - Download original supplier file

- [ ] **AC6:** Export reconciliation report
  - Export reconciliation table as Excel
  - Include: order-level detail + summary
