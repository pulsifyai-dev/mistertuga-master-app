## Story

**As an** admin user,
**I want** daily revenue automatically computed from stored orders and displayed in charts and tables,
**So that** I can track revenue trends without relying on the n8n "Faturamento" workflow.

---

## Context

The n8n "Faturamento e Taxas (Diario)" workflow currently fetches Shopify data and calculates daily revenue/taxes. This story re-implements that logic in-app, using the orders table as source of truth instead of hitting the Shopify API directly. The n8n workflow references: processFeesPercent, cogsPercent, fulfillmentPercent as cost parameters.

**Reference:** `docs/product/mistertuga_master.md` §6.2, n8n workflow "Faturamento e Taxas (Diario)"

---

## Acceptance Criteria

- [ ] **AC1:** Daily revenue table created
  - Migration 023: daily_revenue (date, country, totals)
  - Migration 024: daily_revenue_breakdown (optional per-product breakdown)
  - UNIQUE(date, country_code) — one row per day per country

- [ ] **AC2:** Revenue calculation engine
  - Server Action: `calculateDailyRevenue(date, countryCode?)`
  - Aggregates from orders table: total_revenue, total_tax, total_shipping, order_count, item_count
  - Optional country filter or "all countries" aggregate
  - Idempotent: re-calculating same date overwrites (UPSERT)

- [ ] **AC3:** Automatic daily calculation
  - Triggered at end of day (23:59 UTC) via scheduled function
  - Calculates for all active countries
  - Manual trigger: "Recalculate" button for any date/range

- [ ] **AC4:** Revenue dashboard page
  - Date range selector (default: last 30 days)
  - Revenue chart: line/bar chart by day
  - Revenue table: date, orders, items, revenue, tax, shipping
  - Country filter: all countries or specific
  - Totals row at bottom

- [ ] **AC5:** Profit integration
  - Daily revenue feeds into profit calculation
  - Profit = Daily Revenue - (Ad Spend + Manual Expenses + Service Costs + Supplier Costs)
  - Breakdown visible by: day, week, month

- [ ] **AC6:** Country breakdown view
  - Revenue split by country (pie/bar chart)
  - Top products by revenue (optional)
  - Comparison: this period vs previous period
