# Story 2.6: Expense Catalog + Manual Expenses

**Epic:** EPIC-2 (Post-Migration New Features)
**Sprint:** 5
**Effort:** 1.5 days
**Priority:** P2
**Status:** Draft
**Agent:** @dev (Dex)
**Blocked By:** EPIC-1 complete

---

## Story

**As an** admin user,
**I want** a catalog of services/software with their costs and the ability to add manual expenses,
**So that** the profit dashboard reflects all operational costs including subscriptions and one-off expenses.

---

## Context

The product spec describes a "Services/Softwares" concept for tracking operational expenses like tool subscriptions. This replaces the current simple expenses table with a more structured system: an expense catalog (categories) and manual expense entries linked to those categories.

**Reference:** `docs/product/mistertuga_master.md` §2.2

---

## Acceptance Criteria

- [ ] **AC1:** Expense categories table created
  - Migration 021: expense_categories
  - Fields: name, category type, fixed_monthly_cost, is_active, notes
  - Admin CRUD

- [ ] **AC2:** Manual expenses table created
  - Migration 022: manual_expenses
  - Fields: category_id (optional), description, amount, date, notes, created_by
  - Links to expense_categories for classification

- [ ] **AC3:** Expense catalog management (Settings)
  - List of services/software with: name, category, monthly cost, status
  - Add/edit/remove services
  - Mark active/inactive
  - Optional: fixed monthly cost auto-generates monthly expense entries

- [ ] **AC4:** Manual expense entry
  - Form: select service/software (or "Other"), amount, date, notes
  - Optional categorization
  - Created_by tracked for audit

- [ ] **AC5:** Expense list view
  - Dedicated page or section in Profit module
  - Filter by: date range, category, service
  - Total by month/category
  - Edit/delete existing expenses

- [ ] **AC6:** Profit dashboard integration
  - Manual expenses + fixed monthly costs appear in expense breakdown
  - Total expenses = Ad Spend + Manual Expenses + Fixed Service Costs
  - Profit = Revenue - Total Expenses

- [ ] **AC7:** RLS policies
  - Expense categories: Admin full CRUD, Fornecedor no access
  - Manual expenses: Admin full CRUD, Fornecedor no access
