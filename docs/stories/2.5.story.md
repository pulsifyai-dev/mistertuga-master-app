**Agent:** @dev (Dex)

## Story

**As an** admin user,
**I want** Google Ads and Meta Ads spend data automatically fetched and displayed in the profit dashboard,
**So that** I have a complete view of revenue vs all expenses without checking multiple platforms.

---

## Context

The profit dashboard currently shows Shopify revenue + manual expenses. Ad spend is the largest operational cost and must be integrated for accurate profit calculation. Both platforms offer APIs for fetching daily spend data. The n8n "Faturamento" workflow already references a Facebook Ad Account ID and Shopify store config.

**Reference:** `docs/product/mistertuga_master.md` §2.1-2.3
**Known config:** Meta Ads account ID: referenced in n8n workflow

---

## Acceptance Criteria

- [ ] **AC1:** Ad accounts table created
  - Migration 019: ad_accounts (platform, account_id, config)
  - Support: google_ads, meta_ads
  - API tokens stored in environment variables (NEVER in database)

- [ ] **AC2:** Ad spend table created
  - Migration 020: ad_spend (daily granularity)
  - Columns: date, spend, impressions, clicks, conversions, currency
  - UNIQUE constraint on (ad_account_id, date) — idempotent fetches

- [ ] **AC3:** Google Ads API integration
  - Server Action: `fetchGoogleAdsSpend(accountId, dateRange)`
  - Fetches daily spend from Google Ads Reporting API
  - Stores in ad_spend table
  - Handles pagination, rate limits
  - Error handling: API failure → log error, don't corrupt existing data

- [ ] **AC4:** Meta Ads API integration
  - Server Action: `fetchMetaAdsSpend(accountId, dateRange)`
  - Fetches daily spend from Meta Marketing API
  - Same storage + error handling pattern as Google Ads

- [ ] **AC5:** Automated daily sync
  - Cron job or scheduled function: fetch yesterday's spend at 06:00 UTC
  - Manual trigger: "Sync Now" button in Settings
  - Sync status visible: last sync time, success/failure

- [ ] **AC6:** Profit dashboard integration
  - Ad spend appears as "Google Ads" and "Meta Ads" expense lines
  - Total ad spend aggregated by date range
  - Profit = Shopify Revenue - Ad Spend - Manual Expenses - Other Costs

- [ ] **AC7:** Ad account management UI (Settings)
  - Add/remove ad accounts
  - Test connection button
  - View sync history (last 7 days)
