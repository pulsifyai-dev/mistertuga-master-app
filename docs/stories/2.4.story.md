**Agent:** @dev (Dex)

---

## Story

**As an** admin user,
**I want** to generate supplier export Excel files with embedded product images directly from the app,
**So that** I no longer need the Electron app or Google Drive for supplier order documents.

---

## Context

The current supplier export flow uses n8n + Google Drive to create Excel files for the supplier. The target flow generates the file entirely in-app using ExcelJS, with checkpoint-based export tracking to prevent duplicates. Critical requirement: product images must be embedded in the Excel (not just URLs).

**Reference:** `docs/product/mistertuga_master.md` §5.3-5.5, n8n workflow "Fluxo para Enviar Artigos para o Supplier"
**Export columns:** Order Name, Name of the Item, Quantity, Customer Name, Address, Size, Customization, Version, ThumbnailUrl (embedded)

---

## Acceptance Criteria

- [ ] **AC1:** Supplier exports + items tables created
  - Migration 017: supplier_exports
  - Migration 018: supplier_export_items
  - Tracks export metadata: country, order range, totals, file URL

- [ ] **AC2:** Checkpoint-based export generation
  - Uses export_checkpoints table (created in EPIC-1)
  - "Generate Export" button per country tab
  - Fetches orders from last checkpoint to current
  - Never exports the same order twice
  - Updates checkpoint after successful export

- [ ] **AC3:** Excel file generation with correct columns
  - Columns match reference: Order Name, Name of the Item, Quantity, Customer Name, Address, Size, Customization, Version, Thumbnail
  - Uses ExcelJS (dynamic import — already in Story 1.7 setup)
  - File naming: `{country}_orders_{date}_{orderRange}_{qty}pcs.xlsx`

- [ ] **AC4:** Embedded product images in Excel
  - Fetch thumbnail URLs from order_items
  - Download images and embed as cell images in Excel
  - Image size: max 60x60px thumbnail per row
  - Graceful fallback: if image fetch fails, show URL text instead

- [ ] **AC5:** Export includes cost data
  - Additional columns: Production Cost, Shipping Cost, Total Cost
  - Values from order_costs table (Story 2.3)
  - Summary row at bottom: totals for all cost columns

- [ ] **AC6:** Direct download from app
  - After generation, file stored in Supabase Storage
  - Download button immediately available
  - Export history visible: date, file name, order range, total items
  - Previous exports downloadable from history

- [ ] **AC7:** Multi-country support
  - Works for any country (PT, ES, DE, future countries)
  - Each country has independent checkpoint
  - Country tab shows "X orders ready for export" count

## Testing

- Images embedded in Excel cells (open in Excel/Google Sheets to verify)
